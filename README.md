# Sheka Zone Police Bureau — HRMIS

Advanced Human Resource Management & Payroll Command System.

## Deployment

This is a static single-page application. Deploy to Vercel with:

```bash
vercel --prod
```

## Tech Stack
- Pure HTML / CSS / JavaScript (no framework)
- Chart.js for analytics
- localStorage for client-side data persistence

## Security Note
Admin credentials are NOT stored in plain text in the source code.
The system administrator must securely communicate login credentials
to authorized users through official channels only.
